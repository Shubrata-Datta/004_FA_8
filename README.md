## 📌 Project Overview

This Flutter project demonstrates the use of **Column** and **Row** layouts along with **Container widgets** to build a structured and responsive UI.

The app includes:

* A centered **AppBar title**
* A top row with 3 colored boxes
* A large central container
* A bottom row with left and right aligned containers

---

## 🎯 Features

✅ Clean and simple UI design
✅ Use of `Column` and `Row` layout widgets
✅ Reusable container components
✅ Responsive spacing using `MainAxisAlignment`
✅ Styled containers with colors and rounded corners
